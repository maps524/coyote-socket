# Gamepad Icons

Xbox controller prompt SVGs from **Kenney Input Prompts 1.4.1** (https://kenney.nl/assets/input-prompts).

License: **CC0 1.0 Universal** — public domain, no attribution required (but credited here as courtesy).

Subset: 27 SVGs covering buttons (A/B/X/Y), bumpers (LB/RB), triggers (LT/RT), D-pad directions, stick directions + press, view/menu, and the Series controller silhouette.
